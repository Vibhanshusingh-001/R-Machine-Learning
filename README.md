# R-Machine-Learning



## Popular Machine Learning Packages in **R**

### Core Machine Learning Frameworks
- **caret** – Unified ML training framework
- **tidymodels** – Modern ML ecosystem (recipes, parsnip, workflows)
- **mlr3** – Advanced and extensible ML framework

---

### Classical Machine Learning Algorithms
- **randomForest** – Random Forest
- **e1071** – SVM, Naive Bayes
- **glmnet** – Lasso, Ridge, Elastic Net
- **rpart** – Decision trees
- **nnet** – Neural networks

---

### Boosting & Ensemble Models
- **xgboost**
- **gbm**
- **lightgbm** (R interface)

---

### Deep Learning
- **keras**
- **tensorflow**
- **torch** – R interface for PyTorch

---

### Feature Engineering & Data Processing
- **recipes** – Feature engineering (tidymodels)
- **dplyr** – Data manipulation
- **data.table** – High-performance data handling

---

### Model Evaluation & Visualization
- **pROC** – ROC analysis
- **ROCR** – Classification performance metrics
- **ggplot2** – Data visualization

---

### Bioinformatics / Genomics Machine Learning
- **limma** – Differential expression analysis
- **edgeR** – RNA-seq count modeling
- **DESeq2** – Differential expression analysis
- **mixOmics** – Multi-omics data integration
- **caretEnsemble** – Ensemble learning
